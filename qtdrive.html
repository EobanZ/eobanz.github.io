Qt Google Drive

Mit dieser Anwendung wollte ich herausfinden wie man mithilfe des Qt Frameworks mit Cloud Services wie Google Drive, Onedrive, Dropbox, etc.. interagieren kann. Im laufe der Recherche und Entwicklung habe ich mich dann jedoch nur auf einen Service festgelet: Google Drive. Durch diese späte Entscheidung wurde der Code etwas komplexer als eigentlich nötig. Die wichtigsten Funktionen, die für die Kommunikation mit der Cloud verantwortlich sind befinden sich in den Klassen: Authenticator_GoogleDrive & CloudInterface_GoogleDrive.

Die Anwendung ermöglicht es Dateien per Drag and Drop, in neu erstellte Ordner hochzuladen. Diese Ordner werden im Programm aufgelistet und durch einen Doppelklick kann ein Link zum Teilen abgerufen werden. Die Ordner samt Daten können durch ein Kontextmenü auch wieder gelöscht werden.

Im Source Code finden sich unteranderem Beispiele wie man mit der Qt Bibliothek folges durchführen kann

  • OAuth2 Authentifizierung mihilfe des QOAuth2AuthorizationCodeFlow
  • Uploaden von Daten als resumable-upload
  • Uploaden von Daten als multipart-upload
  • Erstellen von Ordnern
  • Löschen von Ordnern
  • Teilen von Ordnern und Daten
  • Erstellen von API queries.
