Augmented Andventure

Im Rahmen eines Master Praktikums bin ich an der Entwicklung eines Augmented Reality Spiels beteiligt. In dem Spiel sollen auf dem Uni Campus verschiedene Rätsel und Minispiele bewältigt werden. Außerdem soll durch Reinforcement Learning gelernt werden, wann dem Spieler verschiedene Arten von Tipps präsentiert werden sollen. Das Spiel wird für die Hololens 1 in Unity entwickelt.

Rainbow Six: Siege WebRadar Hack

Dieser Hack an dem ich gerade aktiv Arbeite erstellt einen Server, dessen API bei Anfragen, Informationen über alle Spieler und den Spielstatus dies Clients liefert. Diese Infos werden verwendet um mithilfe von JavaScript die Spieler an der Richtigen Stelle auf der aktiven Karte anzuzeigen. Jeder der Verbindung zu dem Server herstellen kann, kann also in einem beliebigen Browser sehen, wo sich alle Spieler befinden. Zurzeit bin ich noch dabei einen Weg zu finden die Weldkoordinaten der Spieler korrekt in 2D-Mapkoordinaten umzurechnen.
