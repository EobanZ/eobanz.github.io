Natives Co-Location Plugin für Unity

Abgabe: 01.08.2022

Die zur Software gehörende Literatur befindet sich zurzeit noch in Arbeit. In dieser Arbeit werden drei möglichkeiten vorgestellt, wie die Tranformation zwischen XR-Geräten ermittelt werden können. Im Plugin wurden diese Methoden dann praktisch umgesetzt. Außerdem wurde ebenfalls auf nativer Ebene ein Netzwerk-System implementiert durch das die Posen der Teilnehmer im lokalen Netzwerk sehr einfach synchronisiert werden können. Die Schnittstelle zur Nutzung int Unity wurde ebenfalls implementiert.
